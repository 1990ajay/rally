Job results

Logs and files:

  • Benchmarking logs console.html
  • Logs of all services logs/
  • Rally files rally-plot/

Results of this job can be obtained in different formats:

  • Rally input task
  • HTML report with graphs $ rally task report
  • Plain text aggregated data $ rally task detailed
  • Plain text aggregated data with detailed iterations $ rally task detailed --iterations-data
  • SLA checks $ rally task sla_check
  • Full result in json $ rally task results

About Rally

Rally is benchmark system for OpenStask:

  • Git repository
  • Documentation
  • How to use Rally (locally)
  • How to add Rally job to your project

Steps to repeat locally:

  1. Fetch rally task from here
  2. Fetch rally plugins from here
  3. Install OpenStack and Rally using this instruction
  4. Unzip plugins and put to .rally/plugins/ directory
  5. Run rally task $ rally task start task.txt
